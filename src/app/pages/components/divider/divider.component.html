Divider

Esta página demonstra o uso do componente <br-divider>, utilizado para criar separações visuais entre conteúdos.

Exemplo básico e modo escuro

O componente divider é uma linha neutra que separa conteúdos, sessões ou temas afins.

Este é um exemplo do divider padrão com estilo solid e espessura small.

Em fundos escuros, o divider mantém sua funcionalidade de separação visual e seu contraste garante boa visibilidade.

Este divider utiliza o modo escuro para adaptar-se ao fundo escuro.

Divider vertical

O divider vertical é útil para criar separações entre elementos que estão dispostos lado a lado.

Esta orientação é eficaz em layouts horizontais.

Em fundos escuros, o divider vertical mantém sua função de separação e o contraste deve garantir que seja visível.

Essa combinação cria uma separação elegante e clara.

Estilo da borda

O divider com estilo solid cria uma linha contínua e consistente.

Este estilo é o padrão e o mais utilizado.

O divider com estilo dashed traz um visual diferenciado.

Útil para destacar quebras menos rígidas na estrutura do conteúdo.

Espessura

Espessura small: separação sutil entre conteúdos relacionados.

Útil quando o distanciamento por si só não é suficiente.

Espessura medium: separação mais visível entre seções.

Equilibra visibilidade e discrição.

Espessura large: divisão forte entre blocos de conteúdo.

Indicado para mudanças estruturais importantes.
