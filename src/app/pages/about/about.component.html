Sobre o Projeto

Nota: Veja abaixo os principais recursos utilizados nesta aplicação.

Este projeto é um showcase completo desenvolvido para demonstrar a utilização de @govbr-ds/webcomponents-angular (v2.0.0-next.37) dentro de uma aplicação Angular 19 usando Angular CLI, aplicando o Design System do Governo.

Tecnologias Utilizadas

Framework Principal

Angular 19 - Framework robusto para construção de aplicações web modernas.

Plataforma completa que oferece CLI, injeção de dependência e arquitetura baseada em componentes.

Documentação Angular

Ferramenta de Build

Angular CLI - Ferramenta oficial para scaffolding, build e testes.

CLI integrada ao framework, facilitando geração de código e otimizações de produção.

Documentação Angular CLI

Design System

Design System (v3.6.1) - Sistema de design para manter consistência visual.

GovBR: padrões, componentes e estilos para experiência consistente.

Documentação GovBR

Web Components

Web Components Angular (v2.0.0-next.37) - Biblioteca oficial GovBR para integrar Web Components com Angular.

Documentação Web Components
